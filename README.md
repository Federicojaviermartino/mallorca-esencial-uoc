# Mallorca Esencial

Sitio web dedicado a tres destinos imprescindibles de Mallorca: Valldemossa, Soller y Palma de Mallorca.

Proyecto desarrollado para la asignatura Herramientas HTML y CSS II de la Universitat Oberta de Catalunya (UOC).

## Requisitos

[Node.js](http://nodejs.org/) >= 18.x

## Instalacion

Clonar el repositorio y ejecutar `npm install`.

## Comandos

| Comando         | Descripcion                                                                 |
| --------------- | --------------------------------------------------------------------------- |
| `npm run dev`   | Inicia el servidor de desarrollo y abre el navegador con recarga en vivo.   |
| `npm run build` | Compila, minifica y optimiza los archivos para produccion en la carpeta `dist/`. |
| `npm run clean` | Elimina la carpeta `dist/` y las carpetas de cache.                         |
